What is a Power-Set?

Power set of ${2^s}$ of set s is the set of all subsets of S. Examples: ${S_1} = $ {a, b} ${2^{S_1}} = $ {{a}, {b}, {ab}, {0}} ${2^2} = 4$ total elements

${S_2} = $ {a, b, c} ${2^{S_2}} = $ {{0}, {a}, {b}, {c}, {a,b}, {a,c}, {b,c}, {a,b,c}, } ${2^3} = 8$ total elements

What are features of deterministic accepters?

  1. Process an input string, which consist of a sequence of symbols
  2. Make transitions from one state to another state (depending on current state and current input symbol).
  3. Have a finite number of internal states.
  4. Produces some output in a slightly different form.

What is DFA?

Deterministic Finite Accepter (or DFA) is a simple automaton finite state accepter (accepts only a finite set of internal states and no other memory).

For a DFA, the term “deterministic” signifies that the automaton has one and only one option at any time (only one relative path).

DFA is a five-tuple. M = (Q, ${\sum}$, ${\sigma}$, ${q_0}$, F) Where:

  1. Q = Finite set of states
  2. ${\sum}$ = Finite set of input alphabet symbols
  3. ${\sigma}$ = Transition function mapping: ${Q * {\sum} -> Q}$
  4. ${q_0 \in Q}$ = is the initial state.
  5. F ${\subset}$ ${Q}$ = set of final states. Has multiple final states.

What is a regular language?

It is a language accepted/recognized by a DFA.

What is NFA?

Another simple automaton finite state accepter but the only difference is that it has several options/relative paths that it can go to--therefore it has choices to pick which path. In a NFA, there can be zero or more transition on an input.

Think of it as if NFA explores all choices (by search and backtrack method) and makes no decision UNTIL all the options have been analyzed.

  • IMPORTANT: nondeterminism is important for simplifying the solution of many problems.

NFA is a five-tuple. M = (Q, ${\sum}$, ${\sigma}$, ${q_0}$, F) Where:

  1. Q = Finite set of states
  2. ${\sum}$ = Finite set of input alphabet symbols (finite)
  3. ${\sigma}$ = Transition function mapping: ${Q * {\sum} -> Q}$
  4. ${q_0 \in Q}$ = is the initial state.
  5. F ${\subset}$ ${Q}$ = set of final states. Has multiple final states.
  6. ${\sigma}$ maps: Q * (${\sum}$ ${\cup}$ {${\lambda}$}) -> ${2^Q}$

2^Q is the powerset.
